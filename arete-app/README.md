# Arête
### *excellence, daily*

A gamified life OS — learn a new word every day, complete your tasks, track your movement, compete with friends, and reflect on your growth. Built with React, Framer Motion, and animated comic characters.

---

## 🚀 Local Setup

```bash
npm install
npm start
```

App runs at `http://localhost:3000`

---

## 📱 Features

- **Home** — Time-aware greeting, swipeable daily story cards, Sol the sun character
- **Learn** — Word of the day with quiz, XP rewards, Sage the owl
- **Do** — Task manager with AI coach nudges, Dash the lightning bolt
- **Move** — Strava-style workout logging with live session timer, Blaze the cheetah
- **Compete** — Friend leaderboard ranked by XP, Rex the king
- **Reflect** — AI-powered journal with Luna the moon

## 🎨 Characters

| Character | Section | Personality |

| ☀️ Sol | Home | Time-aware, warm, energetic |
| 🦉 Sage | Learn | Intellectual, adjusts glasses |
| ⚡ Dash | Do | Bouncy, impatient, celebratory |
| 🐆 Blaze | Move | Speedy, competitive, athletic |

## 🛠 Stack

- React 18
- Framer Motion (animations + transitions)
- React Router v6
- CSS custom properties (theming)
- LocalStorage (persistence)
